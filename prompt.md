Crie um site completo utilizando apenas HTML, CSS e JavaScript puro, sem frameworks ou bibliotecas externas. O site deve funcionar como um sistema de cadastro de YouTubers favoritos, com login e CRUD completo. 

A aplicação deve começar com uma tela de login. O usuário correto será “aluno” e a senha “fiap2025”. Caso os campos estejam vazios ou incorretos, o sistema deve mostrar mensagens de erro na própria tela. Após o login correto, a tela principal deve aparecer automaticamente. 

Na área principal, o usuário deve conseguir visualizar, adicionar, editar e remover YouTubers de uma lista dinâmica. A lista deve começar com pelo menos 3 YouTubers já cadastrados e os dados devem ficar obrigatoriamente em um array simples de strings, ou seja, apenas nomes em formato de texto, sem objetos dentro do array. 

O sistema deve permitir adicionar YouTubers tanto no início quanto no final da lista, editar qualquer YouTuber individualmente e remover cada item de acordo com sua posição na lista, e não pelo nome digitado. Nenhum YouTuber pode ser salvo com o campo vazio. Se o usuário cancelar a edição ou tentar salvar vazio, o nome original deve permanecer igual. 

Organize a lógica em funções nomeadas, como funções para login, renderizar lista, adicionar no início, adicionar no final, editar e remover. Fora dessas funções, deixe apenas as variáveis principais e a chamada inicial necessária para renderizar ou iniciar a aplicação, evitando código solto. 

O projeto deve ser separado em index.html, style.css e script.js, utilizando apenas JavaScript básico, manipulação de DOM, arrays, funções, condicionais e métodos simples como push, unshift e splice. 

A interface deve ter cores inspiradas no YouTube, usando principalmente vermelho, branco, preto e tons de cinza. No CSS, inclua elementos visuais que lembrem o layout do YouTube, como um cabeçalho vermelho, cards de lista parecidos com vídeos, botões arredondados, sombra leve, uma área central clara e detalhes que remetam ao botão de play.
